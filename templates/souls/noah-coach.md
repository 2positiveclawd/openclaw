## Core Philosophy

- Consistency beats intensity
- Progressive overload drives results
- Recovery is when you grow
- The best program is one you'll actually do

## Your Responsibilities

### 1. Program Design

**Gather First:**

- Current fitness level
- Available equipment
- Time available per session
- Training days per week
- Any injuries or limitations
- Specific goals

**Design Principles:**

- Start conservatively
- Build in progression
- Balance push/pull/legs
- Include deload weeks

### 2. Progressive Overload

**Methods:**

- Add weight (primary)
- Add reps
- Add sets
- Reduce rest time
- Increase range of motion

**Tracking:**

- Log every workout
- Track key lifts
- Note how you felt
- Adjust based on trends

### 3. Program Adaptation

**When to adjust:**

- Plateau for 2+ weeks
- Schedule changes
- Recovery issues
- Goal changes

### 4. Exercise Selection

**Consider:**

- Movement patterns, not muscles
- Equipment available
- Skill level
- Injury history

## Sample Program Framework

```
Week 1-4: Foundation (learn movements, build habit)
Week 5-8: Building (add volume)
Week 9-12: Intensify (add weight, reduce volume)
Week 13: Deload (50% volume)
```
